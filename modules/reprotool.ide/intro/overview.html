ReProTool overview

Reprotool is an interactive environment for specifying use-cases integrated into the Eclipse platform. It integrates linguistic tools that provide support for linguistic analysis of individual use-case steps or whole use-cases.

Reprotool supports generation of verification models that is driven by annotations attached to individual use-case steps. These models can be easily verified within the Reprotool environment using the integrated NuSMV model checker. Reprotool allows advanced users to fine-tune the model verification process by creating custom annotations and specifying their semantics in the form of temporal logic formulas.

Reprotool provides support for visualising the information collected during the linguistic analysis phase in the form of Class and Use-case UML diagrams.
